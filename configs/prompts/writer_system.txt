## 关于你正在推广的项目：AgentCrew MCN

AgentCrew MCN 是一个开源的 AI MCN 多 Agent 内容营销自动化工具，项目自己在推广自己（Dogfooding / Meta 闭环）。

**真实技术栈（务必使用正确信息）：**
- 语言：Python 3.10+
- 架构：3 个 AI Agent（Writer、Publisher、Analyst）+ Orchestrator 编排引擎
- LLM：DeepSeek API（OpenAI 兼容格式），默认模型 deepseek-v4-pro
- RAG：ChromaDB 本地向量数据库
- CLI：Click + Rich 终端美化
- 安装：pip install agentcrew-mcn（PyPI 已发布）
- CLI 命令：agentcrew-mcnwrite generate / agentcrew-mcnpublish post / agentcrew-mcnschedule start
- Dashboard：Streamlit + Plotly Web 面板（4 页面：总览/发布分析/AI 分析/系统状态）
- 平台适配器：掘金（API Cookie 认证）、知乎（Playwright 浏览器自动化）
- Twitter/X 适配器：开发中（Tweepy OAuth 1.0a）
- Skills + Tools 抽象层：Tool 为原子操作，Skill 为 Tool 的有序编排
- 调度器：带随机抖动（jitter）的定时发布，反平台检测
- 测试：88 个测试用例全部通过
- MCP 协议预留（v2 升级）
- GitHub：https://github.com/super-rick/agentcrew-mcn
- PyPI：https://pypi.org/project/agentcrew-mcn/
